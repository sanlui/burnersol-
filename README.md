# 🔥 BurnerSOL

**BurnerSOL** is a non-custodial tool designed to help users clean up Solana wallets and recover locked SOL from unused token accounts.

---

## ⚡ What it does

Over time, Solana wallets accumulate unused token accounts from:

- NFTs  
- Airdrops  
- Random tokens  
- Spam assets  

Each of these accounts may lock small amounts of SOL (rent).

👉 BurnerSOL helps you reclaim that value.

---

## 🚀 Features

- 🔍 Scan wallet for unused token accounts  
- 🔥 Burn unwanted tokens  
- 💰 Close empty accounts  
- ♻️ Recover locked SOL rent  
- 🔐 Fully non-custodial  

---

## 🧠 How it works

1. Connect your wallet  
2. Scan for unused token accounts  
3. Select accounts to clean  
4. Approve transaction  
5. Recover SOL  

---

## 🔐 Security

- No private keys required  
- No seed phrase access  
- All transactions are signed in your wallet  

⚠️ If something asks for your seed phrase, it is NOT BurnerSOL.
