🔥 BurnerSOL

Clean your Solana wallet. Recover your SOL.

BurnerSOL is a non-custodial Web3 tool that helps you remove unused token accounts, burn unwanted assets, and reclaim locked SOL rent — safely and directly from your wallet.

---

⚡ Why BurnerSOL

Over time, Solana wallets accumulate:

- 🧩 Empty token accounts
- 🪂 Airdrop spam
- 🎨 Unwanted NFTs
- 🧱 Dust assets

Each of these may lock small amounts of SOL.

👉 BurnerSOL helps you unlock and recover that value.

---

🚀 Features

- 🔍 Wallet scanning for unused token accounts
- 🔥 Token & NFT cleanup
- 💰 SOL rent recovery
- ⚡ Fast on-chain execution
- 🔐 100% non-custodial

---

🧠 How it works

1. Connect your wallet
2. Scan for unused accounts
3. Select what to remove
4. Approve transaction
5. Recover SOL

All actions are executed on-chain and fully transparent.

---

🔐 Security First

BurnerSOL is built with strict security principles:

- No private keys ever requested
- No seed phrase access
- No custody of user funds
- All transactions signed by the user

⚠️ If anything asks for your seed phrase, it is NOT BurnerSOL.

---

📊 Typical Results

After a cleanup:

- 10–20 accounts closed
- 0.01–0.05 SOL recovered

Small amounts — but real value that would otherwise stay locked.
